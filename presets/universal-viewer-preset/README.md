# Universal Viewer Preset

This preset narrows the manifest editor to the fields and creators with clear Universal Viewer support.

## Known Gaps

- Universal Viewer supports PDF/document, EPUB/OPF, 3D/model, and DICOM content types that the manifest editor does not yet author with first-class creators.
- Universal Viewer supports `accompanyingCanvas`, but the manifest editor does not currently expose authoring for it.
- Universal Viewer consumes auth/search/service-driven behavior that the manifest editor does not currently provide strong authoring UI for.
